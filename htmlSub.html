Learning HTML5:

  • What is HTML?
    • HTML stands for Hyper Text Markup Language, this is a coding language that is used to give a webpage structure.
  • In HTML5, what statement is required before the actual HTML document starts?
    • HTML5 requires a declaration statement in the form:
    < !doctype html>
  • An HTML document is divided into two sections generally: head and body. Usually 4 elements are present in the head tags.
    1. Title Tag: Used to name the page.
      2. < title>Title Here< /title>
    2. Meta Tag: Used to provide metadata to search engines.
      3. < meta name="viewport" content="width=device-width">
    3. Meta Charset: Used to determine the chars used in the website.
      4. < meta charset="UTF-8">
    4. Style Tag: Used to add CSS to the page.
      5. < style> CSS Selectors < /style>

    • Div: Changes the line before and after the content.
    Span: A line tag that is used to carry attributes.

    Examples:
    < div> Content Goes Here < /div>
    < span> Content Goes Here < /span>
  • What does the ALT attribute of an image do?
    • If the original image is not available anymore, the page will display the ALT text in its place.
