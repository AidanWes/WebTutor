Learning CSS:

  • What is CSS?
    • CSS stands for Cascading Style Sheet This is a coding language used for adding style to a webpage.
  • General CSS Syntax:
    • selector{cssStatement: value;}
  • 3 Major kinds of CSS Selectors:
    1. Class Selector:
      .className{statement: value;}
    2. Tag Selector:
      tagName{statement: value;}
    3. ID Selector:
      #idName{statements: value;}
  • Three levels of CSS Styling:
    1. Inline CSS: Uses HTML attributes to style an individual element.
    2. Embedded CSS: Places a CSS document in the html document to be used.
    3. External CSS: Defines a CSS document outside of the HTML document to create a seperate file that ends in .css.
  • Properties of CSS:
    • These are styles you can apply to your HTML elements.
